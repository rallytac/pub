Detailed Description

Describes how the Presence is configured for a group of type Group::gtPresence in Group::Type_t.

Helper C++ class to serialize and de-serialize Presence JSON

Example:

{
"forceOnAudioTransmit": false,
"format": 0,
"intervalSecs": 30,
"listenOnly": false
}
