Detailed Description

Custom Audio Device Configuration.

Helper C++ class to serialize and de-serialize AudioDeviceDescriptor JSON used in engageAudioDeviceRegister API.

Example:

{
"boostPercentage": 0,
"channels": 0,
"deviceId": 0,
"direction": 0,
"samplingRate": 0
}
See also
engageAudioDeviceRegister
