Detailed Description

Configuration when using the engageBeginGroupTxAdvanced API.

Helper C++ class to serialize and de-serialize AdvancedTxParams JSON

Example JSON:

{
"alias": "",
"flags": 0,
"includeNodeId": false,
"priority": 0,
"subchannelTag": 0
}
