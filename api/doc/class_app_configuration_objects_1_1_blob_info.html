Detailed Description

Describes the Blob data being sent used in the engageSendGroupBlob API.

Helper C++ class to serialize and de-serialize BlobInfo JSON

Example JSON:

{
"payloadType": 0,
"rtpHeader": {
"marker": false,
"pt": 0,
"seq": 123,
"ssrc": 12345678,
"ts": 87654321
},
"size": 0,
"source": "",
"target": ""
}
