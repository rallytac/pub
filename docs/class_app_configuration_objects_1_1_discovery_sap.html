Detailed Description

Session Announcement Discovery settings settings.

Helper C++ class to serialize and de-serialize DiscoverySap JSON

Example:

{
"address": {
"address": "",
"port": 0
},
"advertising": {
"alwaysAdvertise": false,
"enabled": false,
"intervalMs": 20000
},
"ageTimeoutMs": 30000,
"enabled": false,
"interfaceName": ""
}
