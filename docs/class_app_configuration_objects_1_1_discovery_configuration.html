Detailed Description

Configuration for the Discovery features.

Helper C++ class to serialize and de-serialize DiscoveryConfiguration JSON

Example:

{
"cistech": {
"address": {
"address": "",
"port": 0
},
"ageTimeoutMs": 30000,
"enabled": false,
"interfaceName": ""
