Detailed Description

Location information used as part of the PresenceDescriptor.

Helper C++ class to serialize and de-serialize Location JSON

Location is presented by using the World Geodetic System (WGS84)

Example JSON:

{
"altitude": 123.0,
"direction": 1.0,
"latitude": 123.456,
"longitude": 456.789,
"speed": 1234.0,
"ts": 123456
}
