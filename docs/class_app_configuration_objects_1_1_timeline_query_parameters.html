Detailed Description

Parameters for querying the group timeline.

Helper C++ class to serialize and de-serialize TimelineQueryParameters JSON

Example:

{
"endedOnOrBefore": 0,
"maxCount": 50,
"mostRecentFirst": true,
"onlyAlias": "",
"onlyCommitted": true,
"onlyDirection": 0,
"onlyNodeId": "",
"onlyType": 0,
"onlyTxId": 0,
"startedOnOrAfter": 0
}
